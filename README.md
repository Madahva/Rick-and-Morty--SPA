# Rick and Morty React-app

This is a Full-stack web app, SPA and CRUD with certain features such as: Pagination (infinity scroll), Filter, Fetching Data, Responsive, etc. 

## Built with
- TypeScript
- React
- Redux-toolkit
- CSS Flexbox
- Mobile-first workflow
- API Rest
- Figma
- Node.js
- Express
- Sequelize
- PostgreSQL
